# InsightBoard

InsightBoard is a full-stack MERN analytics dashboard application.

## Features

- User Registration & Login
- JWT Authentication
- Protected Routes
- Analytics Dashboard
- Charts & Statistics
- Responsive UI
- MongoDB Database

## Tech Stack

### Frontend
- React.js
- React Router
- CSS

### Backend
- Node.js
- Express.js

### Database
- MongoDB Atlas

### Authentication
- JWT
- Bcrypt.js
